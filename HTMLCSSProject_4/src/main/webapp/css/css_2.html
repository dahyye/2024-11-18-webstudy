[이코노미스트 라예진 기자] 갤럭시 S25 시리즈가 사전 판매 중 최다 판매 신기록을 세웠다. 삼성전자는 1월 24일부터 2월 3일까지 진행한 갤럭시 S25 시리즈 국내 사전 판매에서 130만대를 기록했다고 4일 밝혔다. 직전까지의 S 시리즈 최다 사전 판매 기록은 갤럭시 S24 시리즈 였으며 지난해 7일간 진행한 사전 판매에서 121만대를 기록했다. 갤럭시 S25 시리즈는 이 기록을 1년만에 갈아치웠다. 이번 사전 판매는 임시공휴일을 포함한 설 연휴 기간이 4일인 점을 감안해 11일동안 운영됐다. 갤럭시 S25 시리즈 사전 판매의 모델별 판매 비중은 갤럭시 S25 울트라 52%, 갤럭시 S25 26%, 갤럭시 S25+ 22%이다. 색상은 갤럭시 S25 울트라는 티타늄 화이트실버, 티타늄 실버블루가 갤럭시 S25+와 갤럭시 S25는 아이스블루와 실버 쉐도우의 인기가 높았다. 삼성닷컴, 삼성 강남 전용 컬러 중에서는 티타늄 제트블랙과 블루블랙의 인기가 높았다.
